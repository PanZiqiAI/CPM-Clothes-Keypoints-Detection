# CPM-Clothes-Keypoints-Detection
Convolutional Pose Machine implemented for clothes key points detection.
## 依赖环境和库
Ubuntu 16.04, Python3.5 with  
* numpy (1.14.2)
* mxnet-cu80 
* matplotlib (2.1.2)
* imageio (2.3.0)
* scikit-image (0.13.1)
* xlrd (1.1.0)
* xlwt (1.3.0)
